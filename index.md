# 🚀 Система учета патентных заявок

**Документация к курсовому проекту**  
*Версия 1.0*

---

## 📌 Быстрый старт
1. **Добавление заявки**: Нажмите кнопку *"Добавить"* в главном окне.
2. **Редактирование**: Выберите заявку и нажмите *"Редактировать"*.
3. **Отчеты**: Для генерации PDF используйте кнопку *"Сформировать отчет"*.
